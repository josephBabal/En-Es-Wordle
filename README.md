<h3 align="center"> Getting Started <h3>

How to run project locally

1. Clone the repository 
``` 
git clone https://github.com/josephBabal/En-Es-Wordle.git
```

2. Install the dependencies
```
cd frontend
npm install
```


3. Run the project
```
cd frontend
npm run dev
```
